## Project Overview

This repository contains my personal identity system created for CMPG214 Assignment 1. The project explores how identity, goals, and values can be translated into a simple visual design.

The goal of the assignment is to design a logomark and supporting infographic that answers the question:

*"If this were the only thing someone saw about me, what would they understand?"*

## Contents

This repository includes:

* My personal **logomark design**
* The final **infographic**
* A **PDF version** of the assignment submission
* Optional **design sketches and iterations**

## Design Idea

My identity design represents growth, ambition, and my focus on technology. The logomark is intentionally simple so that it can be easily redrawn and understood, reflecting clarity and discipline in my approach to problem solving and development.

## Purpose of This Repository

This repository acts as a professional extension of my infographic and demonstrates how my identity and work can be presented clearly and organised within a development environment such as GitHub.
